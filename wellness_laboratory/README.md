# Wellness Hospital - Laboratory Portal

This is the secure management portal for the hospital laboratory staff.

## Features
- Secure authentication for laboratory staff.
- Dashboard for viewing and managing lab test requests.
- Real-time status updates for tests.
- Integration with the hospital's central database.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS V4
- **Database**: Prisma ORM (PostgreSQL)
- **Authentication**: NextAuth.js
- **Icons**: Lucide React

## Getting Started

1. Clone the repository.
2. Navigate to `wellness_laboratory`.
3. Install dependencies: `npm install`.
4. Run development server: `npm run dev`.
5. Open `http://localhost:3005` in your browser.
