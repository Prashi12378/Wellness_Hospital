# Wellness Hospital - Admin Portal

The central control center for Wellness Hospital administrators.

## Features
- **User Management**: Manage doctors, pharmacy staff, and general staff roles.
- **Global Settings**: Configure GST rates, hospital profiles, and logos.
- **Ledger & Accounting**: View detailed financial records across all departments.
- **Staff Management**: Tracking and assignment of staff members.
- **System Reports**: High-level data visualizations of hospital performance.

## Getting Started
1. Install dependencies: `npm install`
2. Sync schema: `npx prisma generate`
3. Run dev server: `npm run dev -p 3002`
