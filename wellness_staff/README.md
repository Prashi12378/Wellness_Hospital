# Wellness Hospital - Staff Portal

Internal operational portal for clinical and non-clinical staff at Wellness Hospital.

## Features
- **Task Management**: Daily operational checklists and task assignments.
- **Patient Coordination**: Real-time status updates on patient flow.
- **Internal Communication**: Messaging and notification system for staff.
- **Attendance & Shifts**: Track work hours and shift schedules.

## Getting Started
1. Install dependencies: `npm install`
2. Sync schema: `npx prisma generate`
3. Run dev server: `npm run dev -p 3004`
