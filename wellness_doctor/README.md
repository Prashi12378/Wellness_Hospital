# Wellness Hospital - Doctor Portal

Clinical interface for healthcare professionals at Wellness Hospital.

## Features
- **Appointment Queue**: Real-time view of daily patient schedules.
- **Digital Prescriptions**: Generate and print prescriptions with medicine database integration.
- **Patient History**: Access past visit records and medical history.
- **Profile Management**: Update professional details and specialization.

## Getting Started
1. Install dependencies: `npm install`
2. Sync schema: `npx prisma generate`
3. Run dev server: `npm run dev -p 3003`
