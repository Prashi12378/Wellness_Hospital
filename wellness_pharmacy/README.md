# Wellness Hospital - Pharmacy Portal

Advanced Billing and Inventory Management system for the hospital's pharmacy.

## Features
- **POS Billing**: Barcode-enabled checkout for medicines.
- **Discounts**: Support for percentage-based and manual discounts.
- **A5 Printing**: GST-compliant invoices optimized for A5 paper.
- **Smart Inventory**: Real-time stock tracking with auto-deduction on sale.
- **History**: Full audit trail of all generated bills.

## Getting Started
1. Install dependencies: `npm install`
2. Sync schema: `npx prisma generate`
3. Run dev server: `npm run dev -p 3001`
