# Wellness Hospital - Patient Portal

The patient-facing side of the Wellness Hospital ERP. This portal allows patients to manage their healthcare journey digitally.

## Features
- **Online Appointment Booking**: Select departments and book visits.
- **My Profile**: Manage personal and contact information.
- **Medical Records**: Access history of visits and digital prescriptions.
- **Lab Reports**: View and track medical test results.
- **Billing History**: Keep track of payments and invoices.

## Getting Started
1. Install dependencies: `npm install`
2. Sync schema: `npx prisma generate`
3. Run dev server: `npm run dev -p 3000`
